第三步:调试

**调试按钮：**
![image-20220301233423496](C:\Users\鹤\AppData\Roaming\Typora\typora-user-images\image-20220301233423496.png)

按钮1：跳转的下一行测试

按钮2:进入自己写的方法体内（不能进入系统定义的方法体）

按钮3：强行进入系统定义的方法体内

按钮4:退出方法体

按钮5：丢帧：重新调试

按钮6:跳转至鼠标光标位置。
